# Gamepad to Mouse

Small utility that implements a virtual mouse that you can control using a gamepad.
Uses SDL to handle the Gamepads currently
Icons provided by [Arks Digital](https://arks.itch.io/) (License [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)):

-   [PlayStation Icons](https://arks.itch.io/)
-   [Xbox Icons](https://arks.itch.io/)

## Current features

-   Configure mouse movement to use either left or right joystick.
-   Set mouse sensitivity.
-   Map mouse right, left and middle click to any gamepad button.
-   Map keyboard arrow keys to any gamepad button.
